Project Name: Data Driven Freamwork
Motivation: Automate login Freamwork

Features:
--------------
a. pull login id and passward from Excel file
b. do the login automation for "http://tutorialsninja.com/demo/index.php?route=account/login"
c. log report on prints.log
d. take snapshot for login failure

Technology used: Selenium WebDriver, TestNg, TestNg Listener, Data driven Freamwork, log4j, page object model, Css selector, Xpath selector, Java.
